Taskman REST API

  • POST /api/register
    • auth type: none
    • req fields: username, password, password2, email
  • POST /api/login
    • auth type: none
    • req fields: username, password
  • POST /api/renew
    • auth type: Bearer token (jwt)
    • req fields: none
  • GET /api/users/current
    • auth type: Bearer token (jwt)
    • req fields: none
  • PUT /api/users/:id
    • auth type: Bearer token (jwt)
    • req fields: username, password, password2, email
  • DEL /api/users/:id
    • auth type: Bearer token (jwt)
    • req fields: none
  • POST /api/lists
    • auth type: Bearer token (jwt)
    • req fields: name, desc, time, priority
  • GET /api/lists
    • auth type: Bearer token (jwt)
    • req fields: none
  • PUT /api/lists/:id
    • auth type: Bearer token (jwt)
    • req: name, desc, time, priority
  • DEL /api/lists/:id
    • auth type: Bearer token (jwt)
    • req: none
  • POST /api/tasks
    • auth type: Bearer token (jwt)
    • req: listid, name, desc, time, priority, state
  • GET /api/tasks
    • auth type: Bearer token (jwt)
    • req fields: none
  • GET /api/tasks/:id
    • auth type: Bearer token (jwt)
    • req fields: none
  • PUT /api/tasks/:id
    • auth type: Bearer token (jwt)
    • req: listid, name, desc, time, priority, state
  • DEL /api/tasks/:id
    • auth type: Bearer token (jwt)
    • req: none
